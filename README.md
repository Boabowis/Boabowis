## SOBRE MÍ

Ingeniera de Software Full Stack con mentalidad orientada a producto y calidad, especializada en el diseño e implementación de aplicaciones móviles multiplataforma altamente escalables y arquitecturas Cloud resilientes. Experiencia demostrable optimizando estructuras pesadas de base de datos para convertirlas en sistemas híbridos ligeros, reduciendo drásticamente los costes operativos y la latencia en la nube. Fiel compromiso con las buenas prácticas, código limpio y mantenible, integración de inteligencia artificial para la generación de contenido dinámico y aplicación rigurosa de protocolos de seguridad en entornos corporativos.

* **Enfoque de Ingeniería de Software:** Aplicación sistemática de patrones de arquitectura, gestión modular del estado e integración asíncrona de APIs RESTful.
* **Arquitectura Cloud y Bases de Datos:** Dominio en optimización de esquemas NoSQL, reglas de seguridad en Firebase (Security Rules) y mecanismos de recuperación de datos híbridos.
* **Mentalidad de Producto:** Focalizada en ofrecer experiencias de usuario de alto impacto, fiabilidad *offline-first* y eficiencia en el backend.

## PROYECTOS DESTACADOS

<details>
<summary><b>[ PROYECTO: ARQUITECTURA HÍBRIDA CLOUD PARA TURISMO ]</b></summary>
<br>

Transformación arquitectónica de nivel empresarial para un sistema móvil internacional de audioguías turísticas, diseñada para resolver cuellos de botella críticos de latencia y sobrecostes en la nube.

| Atributo | Especificación Técnica |
| :--- | :--- |
| **Stack** | Dart, Flutter, Cloud Firestore, Firebase Storage, Python FastAPI |
| **Escala** | Estructura de datos multiidioma con distribución dinámica de recursos localizados |
| **Rendimiento** | Reducción del 70% en lecturas simultáneas de base de datos durante búsquedas de usuarios |
| **Seguridad** | Implementación de Firebase Security Rules personalizadas verificando compras criptográficamente |
| **Impacto** | Reducción radical de costes operativos Cloud y habilitación de navegación completa *offline-first* |
| **Repositorio** | [Ver Estudio Técnico de Arquitectura](https://Boabowis.github.io/portfolio/case_study_turismo.md) |

**Explicación Profesional del Proyecto:**
Liderazgo en la migración desde una estructura NoSQL monolítica hacia una arquitectura de indexación híbrida ligera. Trasladando los payloads estructurales pesados a paquetes JSON estáticos alojados en redes de almacenamiento perimetral (CDN), el sistema garantiza una velocidad de respuesta instantánea incluso bajo conexiones móviles inestables, protegiendo en todo momento los archivos multimedia premium mediante reglas de acceso autenticado.
</details>

Implementación de algoritmos de verificación matricial para evaluar la precisión en la colocación de piezas, retroalimentación auditiva sintética en tiempo real mediante Web Audio API y encapsulamiento visual bajo una interfaz retro-futurista de terminal informático.
